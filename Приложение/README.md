# WB • Отзывы продавца (локальное веб‑приложение)

Это SPA для работы с отзывами Wildberries (категория **Feedbacks and Questions**) с:
- вводом API‑ключа (хранится в **LocalStorage**)
- загрузкой отзывов
- фильтрами/поиском/сортировкой
- кнопками копирования
- полем ответа с **авто‑отправкой** (paste = мгновенно, ручной ввод = debounce/blur)
- обработкой 401/403, 429 (ретраи), 5xx (ошибка + «Повторить загрузку»)
- автообновлением раз в 60 секунд

## 1) Требования

- Windows 10/11
- **Node.js 18+** (рекомендуется LTS)

## Какие токены нужны и какие галочки поставить

В приложении используется **2 токена**:

**1) Токен продавца (отзывы/контент/цены)**
- Категории: **Feedbacks and Questions**, **Content**, **Prices and Discounts**
- Уровень доступа: **Read and Write**

**2) Токен WB Ads (Promotion / реклама)**
- Категория: **Promotion**
- Уровень доступа: **Read only** достаточно (мы читаем кампании и статистику)

Если вы «полный 0» и не хотите писать команды — используйте готовые файлы в корне проекта:

- **ЗАПУСТИТЬ_ДЕСКТОП_В_DEV.bat** — запустит приложение (откроется отдельное окно)
- **СОБРАТЬ_EXE_ДЛЯ_WINDOWS.bat** — соберёт **.exe** (portable) в папку `dist\`

> Важно: Node.js всё равно нужен **1 раз** для сборки. После сборки вы сможете запускать .exe без npm.

## 2) Запуск (режим разработки)

```bash
npm install
npm run dev
```

Откройте адрес, который покажет Vite (обычно `http://localhost:5173`).

## 3) Сборка и запуск как «локальное приложение»

```bash
npm install
npm run build
npm run start
```

После этого открывайте `http://localhost:4173`.

`npm run start` поднимает локальный Express‑сервер для раздачи собранного фронтенда.

## 3.1) Десктоп‑версия (Electron)

### Запуск (без ручных команд)

1) Установите Node.js LTS с https://nodejs.org и перезагрузите компьютер.
2) Двойной клик по файлу **ЗАПУСТИТЬ_ДЕСКТОП_В_DEV.bat**

### Сборка .exe (portable)

1) Установите Node.js LTS и перезагрузите компьютер.
2) Двойной клик по файлу **СОБРАТЬ_EXE_ДЛЯ_WINDOWS.bat**
3) Готовый файл появится в папке **dist\\** (обычно `WB Reviews.exe`)

> В собранной версии внутри приложения поднимается локальный сервер на `http://localhost:4173`.

## 3.1) Десктоп‑режим (Electron)

### Запуск (без сборки .exe)

Двойной клик по **ЗАПУСТИТЬ_ДЕСКТОП_В_DEV.bat**.

### Сборка .exe

Двойной клик по **СОБРАТЬ_EXE_ДЛЯ_WINDOWS.bat**.

Готовый файл появится в папке `dist\`.

## 4) Где в коде эндпоинты WB

Все эндпоинты собраны здесь:
- `src/config.ts` → `WB_ENDPOINTS`

Клиент WB API:
- `src/api/wbClient.ts`

## 5) Примечания

- По умолчанию приложение грузит отзывы за последние **90 дней** (настраивается в фильтре «Период»), чтобы не тянуть всю историю.
- Лимит WB на текст ответа: **2..5000** символов — валидация делается перед отправкой.
